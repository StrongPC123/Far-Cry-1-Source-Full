HBINKTRACK

The basic type HBINKTRACK is a handle to an opened Bink audio track.

Discussion

A HBINKTRACK handle is returned from the BinkOpenTrack function. It also points to a BINKTRACK structure which contains information about the open Bink audio track. If the BinkOpenTrack call failed, then this handle will be zero.


Group: Basic Types
Related Functions: BinkCloseTrack, BinkGetTrackData, BinkOpenTrack
Related Structures: BINKTRACK
