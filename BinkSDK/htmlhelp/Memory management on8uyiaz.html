Memory management on the Nintendo GameCube

Discussion

By default, Bink handles memory management by calling the OS memory allocation functions directly. You can, however, replace the Bink allocation and freeing functions with the RADSetMemory function. You can see how much memory is being used and what the peak use was with the BinkGetSummary function.
