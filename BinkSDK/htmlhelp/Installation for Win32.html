Bink SDK 1.5v

Installation for Win32

Discussion

The Bink installer will be launched automatically when you insert the Bink SDK CD into an auto-play capable machine. If auto-play is not enabled on your machine, you can run the installation program manually by executing Bink.exe in the root directory of the Bink SDK CD.
Once the installer has launched, you will be presented with four installation options. These are:
1) Install Bink SDK, which will install the files you need to build programs that incorporate Bink Video.
2) Install RAD Video Tools, which will install the tools used to compress video files into Bink or Smacker formats.
3) Install Miles Sound System demos, which will install demonstrations of RAD's audio technology product.
4) Install Granny demos, which will install demonstrations of RAD's 3D animation product.
All installation options are largely automatic, and will not require much further user input unless you would like to change an installation parameter such as the installation directory.
