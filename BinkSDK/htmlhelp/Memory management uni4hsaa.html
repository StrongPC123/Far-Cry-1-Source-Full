Memory management under Win32

Discussion

By default, Bink handles memory management by calling the Windows memory functions directly. You can, however, replace the Bink allocation and freeing functions with the RADSetMemory function. You can see how much memory is being used and what the peak use was with the BinkGetSummary function.
