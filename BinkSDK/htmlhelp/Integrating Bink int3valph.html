Integrating Bink into a Win32 Build Environment

Discussion

To build an application that uses Bink Video, you must include the Bink SDK in the build process. There are four simple steps to do this: 1) Add the Bink SDK directory to your include and library directories in your project. 2) Include Bink.h in your source files. 3) Link with the BinkW32.lib import library. 4) Place the BinkW32.DLL file in the directory containing your application. And that's it - you should now be able to use Bink!
