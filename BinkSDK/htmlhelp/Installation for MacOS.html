Bink SDK 1.5v

Installation for MacOS

Discussion

The Bink SDK is provided as a compressed archive in MacBinary format. To install it, simply double-click on the BinkSDK.sit.bin file on the Bink SDK CD. Once installed, you can access this documentation online by clicking on the Bink SDK Help alias in the Bink SDK folder. We've also supplied the RAD Video Tools for Windows on the CD. To compress a file into Bink format, you should install these tools on a Windows machine.
