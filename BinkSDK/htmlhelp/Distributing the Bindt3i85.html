Distributing the Bink Shared Library under MacOS

Discussion

To distribute an application that uses Bink, you must include the Bink Shared Library in the same folder as your application.
Do not place the Bink Shared Library in the System Folder or the Extensions Folder! The Bink Shared Library must be placed in your application folder!
